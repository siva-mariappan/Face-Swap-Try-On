DEEP LEARNING & AI
================================

3. FACE ANALYSIS & RECOGNITION
   - InsightFace 0.7.3
   - Purpose: Face detection, analysis, and feature extraction
   - Models Used:
     * buffalo_l - Face detection suite
     * det_10g.onnx - Face detection (10G parameters)
     * w600k_r50.onnx - Face recognition (ResNet-50 backbone)
     * 1k3d68.onnx - 3D facial landmark detection (68 points)
     * 2d106det.onnx - 2D facial landmark detection (106 points)
     * genderage.onnx - Gender and age estimation

4. FACE SWAPPING MODEL
   - InSwapper (inswapper_128.onnx)
   - Model Size: 529MB
   - Architecture: Generative deep learning model
   - Purpose: High-quality face swapping

5. INFERENCE ENGINE
   - ONNX Runtime 1.18.0
   - Purpose: Optimized model inference
   - Format: ONNX (Open Neural Network Exchange)
   - Execution: CPU-based inference

6. ONNX FRAMEWORK
   - ONNX 1.16.0
   - Purpose: Cross-platform model format
   - Compatibility: Portable across different frameworks
